# Alova Client-Side Skill

Best practices guide for [alova v3](https://alova.js.org/) in browser/client-side/SSR applications.

> For server-side (Node/Bun/Deno), see `alova-server-usage` skill.

## Supported Frameworks

Vue3, Vue2, Nuxt, React, Next.js, React-Native, Expo, Uniapp, Taro, Svelte, SvelteKit, Solid

## Trigger Scenarios

Invoke this skill when working with:

- API requests and data fetching
- alova client setup and initialization
- Cross-component data refresh (`actionDelegationMiddleware` + `accessAction`)
- Request state management and caching
- Business scenarios: paginated lists, form submission, file upload, SSE, etc.

## Documentation Structure

- **SKILL.md** — Quick-reference API index, read this first
- **references/SETUP.md** — Installation and initialization guide

## Usage

For specific API questions, the skill fetches official docs on demand to ensure accuracy.
