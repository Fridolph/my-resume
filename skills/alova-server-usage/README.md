# Alova Server-Side Skill

Best practices guide for [alova v3](https://alova.js.org/) in server-side environments.

> For client-side usage, see `alova-client` skill.

## Supported Environments

Node.js, Bun, Deno, Next.js, Nuxt3, SvelteKit

## Trigger Scenarios

Invoke this skill when working with:

- Server-side API requests and data fetching
- BFF layers, API gateways, microservice forwarding
- SSR data fetching (Next.js app router, Nuxt3 useAsyncData)
- Token authentication on server side
- Distributed caching with Redis

## Documentation Structure

- **SKILL.md** — Quick-reference API index, read this first
- **references/SETUP.md** — Installation and initialization guide
- **references/BFF_API_GATEWAY.md** — BFF/API gateway patterns

## Usage

For specific API questions, the skill fetches official docs on demand to ensure accuracy.
