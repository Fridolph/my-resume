# review-ready 前关键通路校验口袋卡

这张卡只回答一个问题：

> 这轮任务承诺的主线变化，真的打通了吗？

---

## 最短四问

```text
这轮承诺了什么？
真正要过哪几道门？
每道门拿什么证明过了？
现在够不够进入 review-ready？
```
