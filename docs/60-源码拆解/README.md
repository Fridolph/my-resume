# 源码拆解

这里统一存放“结合真实源码的讲解式梳理”。

这一组文档不追求一次性讲完整个系统，而是按“**少量焦点 + 全局视角 + 真实调用链**”逐段拆开，方便后续持续补充。

## 写作约定

- 每篇尽量只聚焦 `3` 个点以内，避免信息过载。
- 优先从“系统全景 → 一条真实链路 → 为什么这样分层”展开。
- 尽量带上流程图、时序图、线框图，帮助建立脑内模型。
- 优先引用当前仓库真实源码位置，而不是抽象空讲。
