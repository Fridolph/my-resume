# 第二阶段执行里程碑

## 说明

第二阶段原始的 P1 已经完成了一次“真实数据链路可行性验证”，但它采用的是过渡方案，不再作为后续正式技术主线继续扩展。

从当前开始，第二阶段的执行计划正式切换到三端统一架构：

- `apps/web`：Nuxt 4 展示端
- `apps/admin`：Nuxt 4 管理后台前端
- `apps/api-server`：NestJS API Server
- 数据层：`SQLite + Drizzle ORM`

因此，下面的里程碑以“统一架构落地”为起点重新规划，并直接覆盖旧计划。

> 这部分工作作为第二阶段的技术探索资产保留，但不再继续扩展为正式后端方案。后续实现统一迁移到 `NestJS + Drizzle ORM`。
